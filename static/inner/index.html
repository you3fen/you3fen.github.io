PROGRAM 01 / RUNNING

bench-scout

A tool for people doing content and e-commerce. See a reference post on your phone, send it over, and it pulls the content from five platforms — WeChat, Bilibili, RED, Douyin, Channels — into a Feishu table you can search and filter.

The earlier version was someone else's, built on Coze in 2025. It sold a couple hundred copies, then got pulled — the cost math didn't work (pay-per-call: the more people used it, the more it lost). I rebuilt it on my own server where the cost is fixed, and added what it was missing: retries, dedup, rate-limiting. It's still running. I use it every day.

About 2,600 lines of Python, 45 tests. I can't write code — I just broke it into steps and had the AI build it, one piece at a time.

PROGRAM 02 / ARCHIVED

clip-montage

An automated pipeline for movie-line supercuts: give it a topic, it searches a line library and cuts a short video. I added a "director" layer — decide what you want first, revise against the footage you can actually get, and if the paper cut doesn't hold up, don't render.

Then I found the free footage only came in 320p — below what I'd accept — so I killed it. I've got this habit: if something doesn't clear my own bar, I'd rather not ship it. Back when I resold clothes hangers, I bought them myself first to check they could hold a heavy coat.
