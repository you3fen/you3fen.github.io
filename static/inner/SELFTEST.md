# 三分内层系统 · 次日人工验收

验收对象：`inner-dev` 分支的 `static/inner/`。当前内容刻意使用占位文案，本清单主要验证结构、交互和边界。

## 1. 准备

- [ ] `git branch --show-current` 输出 `inner-dev`。
- [ ] `git status --short` 无未提交文件。
- [ ] 运行：

  ```bash
  cd ~/sanfen-site
  python3 -m http.server 8766 --bind 127.0.0.1 --directory static
  ```

- [ ] 打开 `http://127.0.0.1:8766/inner/`，控制台无 error 或 warning。

## 2. 开机与系统骨架

- [ ] 首屏显示 `SFBIOS (C)2026 SANFEN`，启动文字逐项出现后自动进入桌面。
- [ ] 启动期间按任意键或点击可跳过。
- [ ] 开始菜单 →“重新启动”可重播引导。
- [ ] 桌面有“关于我、项目、简历、设置、回收站、系统日志”六个原创 CSS 图标。
- [ ] 开始菜单能打开六个应用入口；任务栏时钟显示当前本地时间。

## 3. 窗口引擎

- [ ] 桌面视口下拖动两个窗口，松手后位置保留且不越过桌面上沿。
- [ ] 依次点击重叠窗口，活动标题栏和 z-order 正确切换。
- [ ] 最小化后窗口消失但任务按钮保留；点击任务按钮可恢复。
- [ ] 最大化占满任务栏以上区域，再点一次恢复浮动尺寸。
- [ ] 关闭后窗口与任务按钮都消失；可从桌面图标或开始菜单重新打开。
- [ ] 双击桌面图标可打开应用；单击只选中图标。

## 4. 四个应用与设置

- [ ] “关于我”显示站主占位自述，并能从按钮打开项目和简历。
- [ ] “项目”同时显示 `bench-scout` 与 `clip-montage` 两条占位档案。
- [ ] “简历”显示一页纸式占位内容，明确标注尚未接入 PDF。
- [ ] “设置”切换海松绿、葡萄灰、工业蓝后，桌面和活动标题栏立即换色。
- [ ] 自定义桌面色可用；字号从 12 到 18 像素可调。
- [ ] 刷新后主题、字号和声音开关仍保留；点击“恢复默认”回到海松绿、14 像素、声音开启。

## 5. 回收站彩蛋

- [ ] 回收站只列出两个已删除项目：`TextLens` 与 `clip-montage`。
- [ ] 两项都带停止理由和“已删除”标记。
- [ ] 页面可见文案逐字为：`先给人用,再优化`。
- [ ] 回收站没有真实“还原”或“清空”动作。

## 6. 双环境适配

### 外壳近似屏幕

- [ ] 将内层页面设为约 1248×992：窗口保持浮动、可拖动，桌面和任务栏无横向溢出。
- [ ] 打开全部应用后，任务栏仍能切换窗口。

### 手机直开

- [ ] 以 390×844 打开：活动窗口自动占满任务栏以上区域，不出现页面横向滚动。
- [ ] 开始按钮、内部可滚动的任务列表、时钟三者互不重叠。
- [ ] 连续打开 4 个以上应用，左右滚动任务列表后仍能切换任一窗口。
- [ ] 关于我、项目、简历、设置和回收站内容均可在窗口内部纵向滚动。
- [ ] 手机上最小化、恢复、关闭和开始菜单均可触摸操作。

## 7. 提示音与性能

- [ ] 首次点击后，开始菜单、打开/关闭窗口、最小化/最大化有短促提示音。
- [ ] 设置内取消“启用系统提示音”后不再发声；任务栏扬声器状态同步。重新开启后恢复。
- [ ] Network 面板中过滤 `/inner/`：只有同源 HTML、CSS 和 ES Modules；无 CDN、字体、图片或音频请求。
- [ ] 页面静置一分钟无持续动画、无明显 CPU 占用；系统时钟在分钟变化时更新。
- [ ] `prefers-reduced-motion: reduce` 下引导快速结束，界面仍可用。

## 8. 外壳桥与生产构建

- [ ] 运行 `npm run dev`，按终端实际地址打开完整 3D 外壳。
- [ ] 进入显示器后可操作内层系统；鼠标移动/按下/抬起及键盘按下/抬起没有让外壳进出屏幕交互失效。
- [ ] 运行 `npm run build` 成功；允许出现外壳大体积资源已有的 performance warning，不应有编译 error。
- [ ] `public/inner/` 包含 `index.html`、`styles.css` 与 6 个 JS 文件。
- [ ] `git diff --name-only main...inner-dev` 只涉及 `static/inner/` 与 `CURRENT_STATE.md`，不含 `src/`、Pages workflow、DNS 或域名文件。
